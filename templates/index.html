Kohonen Self-Organising Maps
Pattern recognition and handwritting clustering using the EMNIST database
Kohonen's SOM is a type of artificial neural network that is trained using unsupervised learning to produce a low-dimensional, discretised representation of the input space of the training samples, called a map, and is therefore a method to do dimensionality reduction.
